## Performance Tips / Ù†ØµØ§Ø¦Ø­ Ù„Ù„Ø£Ø¯Ø§Ø¡

1. **Use `will-change` sparingly**
   ```css

     will-change: transform, opacity;
   }
   ```

2. **Prefer `transform` over `position`**
   ```css

   transform: translateY(-10px);

   /* Avoid âœ— */
   ```

3. **Use `backdrop-filter` with caution**
