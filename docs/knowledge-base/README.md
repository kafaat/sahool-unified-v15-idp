## كيفية استخدام هذا الدليل | How to Use This Guide

### للمزارعين | For Farmers
1. **اختر القسم المناسب** من القائمة أعلاه
2. **اقرأ التوصيات** المناسبة لمنطقتك ومحصولك
3. **استخدم تطبيق SAHOOL** لتتبع توصياتك الشخصية

جميع المحتويات متاحة في وضع عدم الاتصال (Offline Mode) عبر تطبيق SAHOOL للجوال.
