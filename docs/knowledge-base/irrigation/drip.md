# الري بالتنقيط | Drip Irrigation

الري بالتنقيط هو أكفأ نظام ري متاح، ومثالي للمناطق شحيحة المياه في الشرق الأوسط.

## نظرة عامة | Overview

### ما هو الري بالتنقيط؟
نظام يوصل المياه مباشرة لمنطقة الجذور عبر نقاطات (Emitters) بمعدلات منخفضة (2-20 لتر/ساعة).

### المميزات | Advantages

| الميزة | الفائدة |
|--------|---------|
| **كفاءة عالية** | 85-95% من المياه تصل للنبات |
| **توفير المياه** | 30-50% مقارنة بالغمر |
| **تقليل الأمراض** | أوراق جافة = [[../diseases/fungal\|فطريات]] أقل |
| **توفير العمالة** | أتمتة كاملة ممكنة |
| **تسميد متكامل** | Fertigation |
| **أي تضاريس** | يعمل على المنحدرات |

### العيوب | Disadvantages

| العيب | الحل |
|-------|------|
| تكلفة أولية عالية | تقسيط، دعم حكومي |
| انسداد النقاطات | فلترة جيدة، صيانة |
| تراكم الأملاح | غسيل دوري |
| تلف من القوارض | حماية الخطوط |

## مكونات النظام | System Components

### المخطط العام | System Layout

```
مصدر المياه
    ↓
━━━━━━━━━━━━━
  وحدة التحكم
  (Head Unit)
━━━━━━━━━━━━━
    ↓
خطوط رئيسية (Main Lines)
    ↓
خطوط فرعية (Sub-main)
    ↓
خطوط التنقيط (Laterals)
    ↓
النقاطات (Emitters)
```

### وحدة التحكم | Head Unit

| المكون | الوظيفة | الأهمية |
|--------|---------|---------|
| **مضخة** | رفع الضغط | أساسي |
| **فلتر رملي** | إزالة الرمل | ضروري للآبار |
| **فلتر قرصي** | إزالة الشوائب الدقيقة | ضروري |
| **فلتر شبكي** | حماية إضافية | موصى |
| **منظم ضغط** | ثبات الضغط | مهم |
| **خزان أسمدة** | للفرتيجيشن | اختياري |
| **عداد مياه** | قياس الاستهلاك | موصى |

### خطوط التنقيط | Drip Lines

#### أنواع خطوط التنقيط

| النوع | الاستخدام | العمر |
|-------|-----------|-------|
| **خراطيم PE** | دائم، أشجار | 10-15 سنة |
| **شرائط رقيقة** | موسمية، خضروات | 1-3 مواسم |
| **نقاطات خارجية** | تعديل قديم | حسب الجودة |
| **خراطيم مدمجة** | استخدام عام | 5-10 سنوات |

#### مواصفات الاختيار

| المحصول | قطر الخرطوم | تصريف النقاط | المسافة |
|---------|-------------|--------------|---------|
| [[../crops/dates\|نخيل]] | 16-20 مم | 8-16 لتر/ساعة | 50-100 سم |
| فواكه | 16 مم | 4-8 لتر/ساعة | 50-75 سم |
| خضروات | 16 مم | 2-4 لتر/ساعة | 20-30 سم |

## تصميم النظام | System Design

### خطوات التصميم | Design Steps

#### 1. تحليل الموقع
- مساحة الحقل
- مصدر المياه (كمية، ضغط، جودة)
- نوع المحصول
- نوع التربة
- الطبوغرافيا

#### 2. حساب الاحتياجات

```
إجمالي الاحتياج (لتر/يوم) =
    عدد النباتات × احتياج النبات × معامل الأمان

مثال لـ 100 نخلة:
= 100 × 200 لتر × 1.2 = 24,000 لتر/يوم
```

#### 3. اختيار المكونات

**تصريف المضخة**:
```
تصريف المضخة (م³/ساعة) =
    الاحتياج اليومي ÷ ساعات التشغيل

= 24 م³ ÷ 6 ساعات = 4 م³/ساعة
```

**ضغط التشغيل**:
- النقاطات: 1-1.5 بار
- فقد الاحتكاك: حسب طول الخطوط
- ارتفاع: 0.1 بار لكل متر ارتفاع

#### 4. رسم المخطط
- خطوط رئيسية وفرعية
- مواقع الفلاتر والصمامات
- توزيع خطوط التنقيط

## تركيب النظام | Installation

### الأدوات المطلوبة | Required Tools

- قاطع أنابيب
- مثقاب للنقاطات
- شريط تفلون
- مشابك
- مقياس ضغط

### خطوات التركيب | Installation Steps

1. **تجهيز وحدة التحكم**
   - تركيب المضخة
   - توصيل الفلاتر بالترتيب
   - تركيب منظم الضغط

2. **مد الخطوط الرئيسية**
   - عمق 30-40 سم للحماية
   - صمامات عند كل قطاع

3. **مد الخطوط الفرعية**
   - موازية لصفوف النباتات
   - صمامات للتحكم

4. **تركيب خطوط التنقيط**
   - على سطح التربة أو مدفونة قليلاً
   - ربط بالخطوط الفرعية

5. **الاختبار**
   - فحص التسريبات
   - قياس الضغط
   - التأكد من توزيع المياه

## تشغيل النظام | Operation

### البدء اليومي | Daily Start

1. فتح صمام المياه الرئيسي
2. تشغيل المضخة
3. فحص الضغط (1-1.5 بار)
4. فتح صمامات القطاعات بالتتابع
5. فحص عمل النقاطات

### الإيقاف | Shutdown

1. إغلاق صمامات الأسمدة (إن وجدت)
2. تشغيل لمدة 5 دقائق للغسيل
3. إغلاق صمامات القطاعات
4. إيقاف المضخة
5. إغلاق الصمام الرئيسي

### جدول التشغيل النموذجي | Typical Schedule

| الموسم | التكرار | المدة |
|--------|---------|-------|
| الشتاء | كل 3-4 أيام | 2-3 ساعات |
| الربيع | كل 2-3 أيام | 3-4 ساعات |
| الصيف | يومياً | 4-6 ساعات |
| الخريف | كل 2-3 أيام | 3-4 ساعات |

> راجع [[scheduling|جدولة الري]] لتفاصيل أكثر

## الصيانة | Maintenance

### الصيانة الدورية | Regular Maintenance

#### يومياً
- [ ] فحص الضغط
- [ ] فحص النقاطات عشوائياً
- [ ] ملاحظة أي تسريب

#### أسبوعياً
- [ ] غسيل نهايات الخطوط (Flushing)
- [ ] تنظيف الفلاتر
- [ ] فحص خزان الأسمدة

#### شهرياً
- [ ] فحص شامل للنقاطات
- [ ] معايرة منظم الضغط
- [ ] فحص الوصلات

#### موسمياً
- [ ] تنظيف كيميائي للنظام
- [ ] استبدال القطع التالفة
- [ ] معايرة النظام

### معالجة الانسداد | Clogging Treatment

#### أنواع الانسداد

| النوع | السبب | العلاج |
|-------|-------|--------|
| **فيزيائي** | رمل، طين | فلترة أفضل، غسيل |
| **كيميائي** | أملاح، حديد | حمض، غسيل |
| **بيولوجي** | طحالب، بكتيريا | كلور، تنظيف |

#### الغسيل الكيميائي

**حمض الفوسفوريك** (للأملاح):
```
تركيز: 0.5-1%
المدة: 30-60 دقيقة
غسيل بعدها: 15 دقيقة ماء نظيف
```

**هيبوكلوريت الصوديوم** (للطحالب):
```
تركيز: 50-100 ppm كلور
المدة: 30 دقيقة
غسيل بعدها: جيد بالماء النظيف
```

## التسميد مع التنقيط (Fertigation) | Fertigation

### المبدأ | Principle
حقن الأسمدة الذائبة مع مياه الري.

### المعدات | Equipment

| المعدة | الاستخدام | التكلفة |
|--------|-----------|---------|
| خزان فنشري | بسيط، صغير | منخفضة |
| مضخة حقن | دقيق، متوسط | متوسطة |
| نظام تناسبي | آلي، كبير | عالية |

### برنامج نموذجي للفرتيجيشن

#### للنخيل ([[../crops/dates|التمور]])

| الشهر | السماد | الكمية/شجرة |
|-------|--------|-------------|
| فبراير-أبريل | نترات البوتاسيوم | 500 جم/أسبوع |
| مايو-يوليو | يوريا + بوتاسيوم | 300+200 جم |
| أغسطس-سبتمبر | سلفات البوتاسيوم | 400 جم/أسبوع |

### نصائح الفرتيجيشن

1. **ابدأ بماء نظيف** - 5 دقائق
2. **حقن السماد** - 80% من وقت الري
3. **اغسل بماء نظيف** - آخر 15 دقيقة
4. **لا تخلط أسمدة متعارضة**
5. **تأكد من ذوبان كامل**

### التكلفة التقديرية لهكتار واحد (نخيل)

| البند | التكلفة (تقريبية) |
|-------|------------------|
| وحدة التحكم | 2000-5000 ريال |
| خطوط رئيسية | 500-1000 ريال |
| خطوط تنقيط | 1500-3000 ريال |
| نقاطات ووصلات | 500-1000 ريال |
| تركيب | 500-1000 ريال |
| **الإجمالي** | **5000-11000 ريال** |

### العائد على الاستثمار | ROI

| البند | الغمر | التنقيط | الفرق |
|-------|-------|---------|-------|
| استهلاك المياه | 100% | 50-60% | توفير 40-50% |
| الإنتاجية | 100% | 120-140% | زيادة 20-40% |
| فاتورة المياه | 100% | 50-60% | توفير كبير |

> استرداد التكلفة خلال 2-3 سنوات عادة
