# Legacy / Archived Versions

هذا المجلد يحتوي على نسخ تاريخية من المشروع للرجوع إليها فقط.

## تحذير هام

**لا تستخدم هذه الملفات في البناء أو CI/CD.**

مصدر الحقيقة الوحيد (Source of Truth) هو:
- `kernel/services/` - الخدمات الحالية
- `shared/` - المكتبات المشتركة
- `docker/` - إعدادات Docker
- `helm/` - Helm Charts

## سياسة الأرشفة

1. أي نسخة جديدة تُنقل هنا يجب توثيقها في الجدول أعلاه
2. يُفضل تحويل النسخ المهمة إلى GitHub Releases
3. هذا المجلد مستثنى من CI/CD
4. للحذف النهائي: تأكد من وجود Release مع Assets

## للمطورين

إذا كنت تبحث عن كود قديم:
1. راجع `git log` أولاً
2. استخدم `git show <commit>:<file>` لعرض ملف من commit قديم
3. إذا لم تجد، ابحث هنا
